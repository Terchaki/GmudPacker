Tutorial de Uso - GMUD Packer

1. Como Usar o Transformador

O gmudPacker foi desenvolvido para facilitar a conversão de arquivos do pipeline de build em estruturas organizadas para empacotamento GMUD.

Passo a Passo:

  1. Acesse a aba "Transformar" na navegação superior
  2. Cole o conteúdo do arquivo log do pipeline na área de texto
  3. Clique no botão "Gerar" para processar o arquivo
  4. Aguarde o processamento (a barra de progresso indicará o status)
  5. Após a conclusão, o resultado aparecerá na área abaixo
  6. Clique em "Copiar" para copiar o resultado processado

2. Formato de Entrada Esperado

O arquivo do pipeline deve conter os arquivos minificados do build em um formato estruturado. O transformador lerá este conteúdo e reorganizará em uma estrutura adequada para o pacote GMUD.

3. Formato de Saída

O resultado será uma listagem estruturada e organizada dos arquivos, pronta para ser utilizada na geração do pacote GMUD. Você poderá copiar diretamente para usar em seu processo de empacotamento.

4. Dicas Úteis

  • Sempre verifique se o conteúdo foi colado completamente antes de processar
  • Use a função "Limpar" para resetar a tela e processar um novo arquivo
  • O progresso é uma estimativa visual do processamento em andamento
  • O resultado pode ser copiado em um clique

5. Suporte e Dúvidas

Em caso de dúvidas ou problemas, verifique se:

  • O arquivo foi copiado completamente do pipeline
  • O formato do arquivo está correto
  • Não há caracteres especiais ou codificação diferente
